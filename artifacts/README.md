# Artifacts

This folder stores supporting experiment notebooks that I kept outside the main project root so the repository stays easier to read.

## What is kept here

- `pytorch_feature_engineering/`: PyTorch notebooks centered on feature-engineering experiments
- `pytorch_separate_added_features/`: a separate PyTorch branch with added feature experiments
- `pytorch_transfer_added_features/`: a transfer-style PyTorch branch with added feature experiments
- `tabnet_experiment/`: TabNet training and inference notebooks

These notebooks are kept here as supporting experiment history, while the main project root contains the cleaner primary workflow.
